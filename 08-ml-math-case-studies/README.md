# ML Math Case Studies - Hands-On Notebooks

This section of the repository contains **Jupyter notebooks** focused on applying **mathematical concepts** to real **Machine Learning algorithms**.  
These notebooks take you through **step-by-step derivations, implementations, and visualizations** of fundamental ML methods using Python.  

The goal is to **bridge the gap between theory and practice**, helping you understand how math drives ML algorithms.

---

```

├── pca-from-scratch.ipynb

└── README.md
```


---

## Notebook Overview

| Notebook | Description |
|----------|-------------|
| `linear-regression-from-scratch.ipynb` | Implement Linear Regression using Python and understand the math behind ordinary least squares. |
| `logistic-regression-from-scratch.ipynb` | Step-by-step implementation of Logistic Regression, including cost function and gradient derivation. |
| `gradient-descent-visualization.ipynb` | Visualize how Gradient Descent works in 1D and 2D to minimize loss functions. |
| `pca-from-scratch.ipynb` | Principal Component Analysis explained mathematically and implemented from scratch for dimensionality reduction. |
| `neural-network-math.ipynb` | Mathematical foundations of neural networks: forward pass, backpropagation, and weight updates. |

---

## Features

- Step-by-step **mathematical derivations** of ML algorithms.
- Practical **Python implementations** without using high-level libraries.
- Visualizations for **intuition and concept clarity**.
- Focus on connecting **linear algebra, calculus, and statistics** to ML.
- Beginner to intermediate-friendly.

## How to Use

1. Clone the repository:

```bash
git clone https://github.com/yourusername/math-for-machine-learning.git
```

2. Navigate to this folder:
```bash
cd 08-ml-math-case-studies
```

3. Open Jupyter Notebook:
```bash
jupyter notebook
```

4. Run notebooks sequentially for best understanding.
