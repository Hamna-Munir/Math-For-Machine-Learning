# Statistics for Machine Learning - Hands-On Notebooks

This folder contains **Jupyter notebooks** focused on the statistical concepts required for **Machine Learning and Data Science**. Statistics helps us analyze data, estimate parameters, test hypotheses, and understand patterns within datasets.

These notebooks combine **mathematical intuition, visual explanations, and Python implementations** to make statistical concepts easier to understand and apply in real-world Machine Learning tasks.

## Notebook Overview

| Notebook | Description |
|----------|-------------|
| `01-descriptive-statistics.ipynb` | Measures such as mean, median, mode, variance, and standard deviation. |
| `02-sampling-methods.ipynb` | Different sampling techniques used in statistics and data analysis. |
| `03-parameter-estimation.ipynb` | Methods for estimating population parameters from sample data. |
| `04-hypothesis-testing.ipynb` | Statistical hypothesis testing and decision making. |
| `05-confidence-intervals.ipynb` | Understanding confidence intervals and uncertainty estimation. |
| `06-correlation-vs-covariance.ipynb` | Relationship between variables using covariance and correlation. |
| `07-bias-variance-tradeoff.ipynb` | Understanding model complexity, bias, variance, and overfitting. |
| `08-statistics-in-ml.ipynb` | How statistical concepts are applied in Machine Learning models. |

## How to Use

1. Clone the repository:

```bash
git clone https://github.com/yourusername/math-for-machine-learning.git
```

2. Navigate to the statistics folder:
   
```bash
cd 04-statistics
```

3. Launch Jupyter Notebook:

```bash
jupyter notebook
```

4. Open notebooks in order and run each cell to follow the learning path.
