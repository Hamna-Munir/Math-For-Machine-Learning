# Calculus for Machine Learning - Hands-On Notebooks

This repository contains a collection of **Jupyter notebooks** focused on teaching **calculus concepts** with practical examples. These notebooks are designed to help students and self-learners understand how calculus is used in **training machine learning models** and optimizing algorithms.

The content connects mathematical theory with real-world applications in **Artificial Intelligence, Data Science, and Deep Learning**.

## Notebook Overview

| Notebook | Description |
|----------|-------------|
| `01-functions-and-limits.ipynb` | Introduction to functions and limits. |
| `02-derivatives-basics.ipynb` | Fundamentals of derivatives and rates of change. |
| `03-rules-of-differentiation.ipynb` | Rules for differentiating complex functions. |
| `04-partial-derivatives.ipynb` | Partial derivatives for multivariable functions. |
| `05-gradients.ipynb` | Gradients and direction of steepest descent. |
| `06-chain-rule.ipynb` | Chain rule and its role in backpropagation. |
| `07-multivariable-functions.ipynb` | Working with functions of multiple variables. |
| `08-hessian-matrix.ipynb` | Hessian matrix and second-order derivatives. |
| `09-taylor-series.ipynb` | Function approximation using Taylor series. |
| `10-integrals-overview.ipynb` | Integration and continuous accumulation. |
| `11-calculus-in-ml.ipynb` | Applications of calculus in machine learning. |

## How to Use

1. Clone this repository:

```bash
git clone https://github.com/yourusername/math-for-machine-learning.git
```

2. Navigate to the folder:

```bash
cd 02-calculus
```

3. Open Jupyter Notebook:

```bash
jupyter notebook
```

4. Open notebooks in order and run each cell.

---

## Requirements

- Python 3.7+
- Jupyter Notebook or JupyterLab
- NumPy
- Matplotlib
- SymPy
- SciPy (for numerical integration)

Install required packages:

```bash
pip install numpy matplotlib sympy scipy jupyter
```
