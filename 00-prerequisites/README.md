# Python & Mathematical Foundations for Machine Learning

This folder provides the foundational building blocks required to understand and implement mathematics for Machine Learning using Python. It is designed to establish clarity between **mathematical concepts, notation, and their practical implementation in code**.

The content is structured to ensure a smooth transition from basic Python-based mathematical operations to NumPy-powered numerical computing, preparing learners for advanced topics such as Linear Algebra, Calculus, and Optimization.

---

## Folder Contents

### 1. `01-python-for-math.ipynb`
Introduces Python as a tool for mathematical computation.
- Basic arithmetic operations
- Mathematical expressions in Python
- Simple function visualization
- Relationship between Python math and Machine Learning algorithms

### 2. `02-numpy-basics.ipynb`
Covers core NumPy concepts essential for numerical computing.
- NumPy array creation and manipulation
- Vectorized operations
- Performance advantages over Python lists
- Role of NumPy in Machine Learning pipelines

### 3. `03-math-notation-guide.md`
A reference guide for mathematical notation commonly used in Machine Learning.
- Scalars, vectors, and matrices
- Summations and norms
- Probability and calculus notation
- Symbols frequently used in ML equations and research papers

---

## Learning Objectives

By completing this folder, you will be able to:
- Use Python confidently for mathematical computations
- Understand why NumPy is essential for Machine Learning
- Read and interpret mathematical notation used in ML
- Connect mathematical expressions with Python implementations
