# Optimization for Machine Learning - Hands-On Notebooks

This repository section contains a collection of **Jupyter notebooks** focused on **optimization techniques** used in Machine Learning.

Optimization is the core process behind **training models**, where we minimize loss functions and improve model performance.

These notebooks bridge the gap between **mathematical intuition** and **practical ML implementation**.

## Notebook Overview

| Notebook | Description |
|----------|-------------|
| `01-optimization-intuition.ipynb` | Intuition behind optimization and why it matters in ML. |
| `02-convex-vs-nonconvex.ipynb` | Difference between convex and non-convex functions. |
| `03-loss-functions.ipynb` | Common loss functions used in Machine Learning. |
| `04-gradient-descent.ipynb` | Gradient Descent algorithm for minimizing loss. |
| `05-stochastic-gradient-descent.ipynb` | SGD and mini-batch learning techniques. |
| `06-momentum-rmsprop-adam.ipynb` | Advanced optimizers (Momentum, RMSProp, Adam). |
| `07-learning-rate-strategies.ipynb` | Learning rate tuning and scheduling methods. |
| `08-regularization.ipynb` | Techniques to prevent overfitting (L1, L2, Dropout). |
| `09-optimization-in-ml.ipynb` | Real-world applications of optimization in ML. |
