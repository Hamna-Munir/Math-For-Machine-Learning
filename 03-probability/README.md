# Probability for Machine Learning - Hands-On Notebooks

This section of the repository contains **Jupyter notebooks** focused on learning **probability theory** and understanding how uncertainty is modeled in **Machine Learning**.

Probability plays a crucial role in many ML algorithms including **Naive Bayes, Bayesian models, probabilistic reasoning, and statistical learning**. These notebooks explain probability concepts step-by-step with practical Python examples.

The goal is to build strong intuition about **randomness, distributions, and statistical reasoning** used in modern AI systems.

## Notebook Overview

| Notebook | Description |
|----------|-------------|
| `01-probability-basics.ipynb` | Introduction to probability concepts, sample spaces, and events. |
| `02-random-variables.ipynb` | Understanding discrete and continuous random variables. |
| `03-probability-distributions.ipynb` | Probability distribution functions and their properties. |
| `04-expected-value-variance.ipynb` | Mean, expectation, variance, and spread of distributions. |
| `05-joint-conditional-probability.ipynb` | Joint probability, conditional probability, and independence. |
| `06-bayes-theorem.ipynb` | Bayes' theorem and updating beliefs using evidence. |
| `07-common-distributions.ipynb` | Important distributions such as Bernoulli, Binomial, and Normal. |
| `08-law-of-large-numbers.ipynb` | Understanding how averages converge with larger samples. |
| `09-central-limit-theorem.ipynb` | Why many distributions approach a normal distribution. |
| `10-probability-in-ml.ipynb` | Practical applications of probability in Machine Learning models. |
