# Information Theory for Machine Learning - Hands-On Notebooks

This section of the repository contains **Jupyter notebooks** focused on learning **Information Theory** and understanding how **uncertainty and information** are quantified in **Machine Learning**.  
Information theory is fundamental to many ML concepts including **entropy, cross-entropy, KL-divergence, mutual information, and model evaluation**. These notebooks explain concepts step-by-step with practical Python examples.

The goal is to build strong intuition about **information, uncertainty, and probabilistic reasoning** used in modern AI systems.

## Notebook Overview

| Notebook | Description |
|----------|-------------|
| `01-entropy.ipynb` | Introduction to **entropy**, measuring uncertainty in random variables. |
| `02-cross-entropy.ipynb` | Understanding **cross-entropy**, commonly used as a loss function in ML. |
| `03-kl-divergence.ipynb` | Kullback-Leibler divergence to measure difference between probability distributions. |
| `04-mutual-information.ipynb` | Measures shared information between variables for feature selection and analysis. |
| `05-information-theory-in-ml.ipynb` | Applications of information theory concepts in Machine Learning models. |
