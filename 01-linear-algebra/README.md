# Linear Algebra - Hands-On Notebooks

This repository contains a series of **Jupyter notebooks** designed to teach and explore **Linear Algebra concepts** with practical examples. These notebooks are especially useful for students, self-learners, and anyone preparing for applications of linear algebra in **Machine Learning** and **Data Science**.

## Notebook Overview

| Notebook | Description |
|----------|-------------|
| `01-scalars-vectors-matrices.ipynb` | Introduction to scalars, vectors, and matrices. |
| `02-vector-operations.ipynb` | Basic vector operations like addition, subtraction, dot product, and norm. |
| `03-matrix-operations.ipynb` | Matrix operations including addition, multiplication, and transpose. |
| `04-linear-combinations.ipynb` | Understanding linear combinations of vectors. |
| `05-span-basis-dimension.ipynb` | Concepts of span, basis, and dimension of vector spaces. |
| `06-linear-independence.ipynb` | Learn about linearly independent and dependent vectors. |
| `07-matrix-rank.ipynb` | Matrix rank and its importance in solving systems of equations. |
| `08-determinant.ipynb` | Determinants of matrices and their properties. |
| `09-inverse-matrix.ipynb` | Computing matrix inverses and their applications. |
| `10-eigenvalues-eigenvectors.ipynb` | Eigenvalues, eigenvectors, and their significance. |
| `11-diagonalization.ipynb` | Diagonalizing matrices and simplifying computations. |
| `12-positive-definite-matrices.ipynb` | Positive definite matrices and their properties. |
| `13-singular-value-decomposition.ipynb` | Singular Value Decomposition (SVD) explained. |
| `14-projections-and-orthogonality.ipynb` | Vector projections, orthogonal vectors, and subspaces. |
| `15-linear-algebra-in-ml.ipynb` | Applications of linear algebra in machine learning. |

## How to Use

1. Clone this repository:

```bash
git clone https://github.com/yourusername/01-linear-algebra.git
```

2. Navigate to the folder:

```bash
cd 01-linear-algebra
```

3. Open a Jupyter notebook:

```bash
jupyter notebook
```

4. Start exploring the notebooks one by one.

## Requirements

- Python 3.7+
- Jupyter Notebook or JupyterLab
- NumPy
- Matplotlib (optional for visualizations)

Install required packages using:

```bash
pip install numpy matplotlib jupyter
```
