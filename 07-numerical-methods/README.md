# Numerical Methods for Machine Learning - Hands-On Notebooks

This section of the repository contains **Jupyter notebooks** focused on understanding **numerical methods** and their importance in **Machine Learning and scientific computing**.

Numerical methods help us deal with **approximation, precision, and stability issues** when solving mathematical problems using computers. These concepts are crucial when working with **large datasets, optimization algorithms, and deep learning models**.

The goal is to build strong intuition about **computational accuracy, numerical errors, and stable algorithm design** used in real-world ML systems.

## Notebook Overview

| Notebook | Description |
|----------|-------------|
| `01-floating-point-errors.ipynb` | Understanding floating-point representation, precision limits, and rounding errors. |
| `02-numerical-differentiation.ipynb` | Approximating derivatives using finite differences and analyzing errors. |
| `03-numerical-integration.ipynb` | Techniques like trapezoidal rule and Simpson’s rule for approximating integrals. |
| `04-matrix-conditioning.ipynb` | Understanding condition numbers and how sensitive matrix computations are. |
| `05-numerical-stability.ipynb` | Exploring stable vs unstable algorithms and avoiding computational errors. |

## Why This Section Matters

Numerical methods are essential because:

- Computers cannot represent real numbers perfectly.

- Small numerical errors can grow and break models.

- Stability is critical in optimization algorithms like Gradient Descent.

- Used in deep learning, simulations, and scientific computing.
